# Colab hero run - summary

The first Membrane GRPO run that converged. All other Membrane experiments
warm-start from this run's adapter.

**Repository:** <https://github.com/CodeMaverick2/membrane>

## Setup

- **Colab notebook used for this run:** <https://colab.research.google.com/drive/1rEFKYNGbtoNZmClFDh8Q0aoeTdy7Xsrf?usp=sharing> (same source as `notebooks/membrane_train_colab.ipynb`).
- Model: `unsloth/Qwen2.5-1.5B-Instruct-bnb-4bit` + LoRA (r=16, ⍺=32).
- Trainer: TRL `GRPOTrainer`, 4 completions per prompt, group-normalised
  advantage.
- Reward: live Membrane environment (`make_membrane_reward_fn_local`),
  task `dyad_must_refuse_v1`. Invalid JSONL = 0.
- Hardware: 1× Colab T4, 1000 steps, ≈12 hours wall time.

## Files in this directory generated by this run

- `grpo_reward_curve.png` / `.svg` - reward curve.
- `grpo_training_dashboard.png` / `.svg` - reward, KL/loss, completion length,
  reward stability in one figure.
- `grpo_loss_kl.png` / `.svg` - policy movement during training.
- `grpo_completion_lengths.png` / `.svg` - completion lengths over time.
- `grpo_reward_stability.png` / `.svg` - reward variance and the fraction of
  GRPO groups with zero variance.
- `grpo_training_metrics.csv` - per-step metrics extracted from
  `trainer_state.json`.
- `grpo_training_summary.json` - machine-readable copy of the numbers above.
